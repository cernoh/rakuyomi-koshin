    **`backend/server/src/track/auth/mal.rs`** — MalAuth (PKCE flow)

    Struct:
    ```rust
    pub struct MalAuth {
        client_id: String,
        client: reqwest::Client,
    }

    impl MalAuth {
        pub fn new() -> Self {
            Self {
                client_id: "c46c9e24640a64dad5be5ca7a1a53a0f".to_string(),  // per PROJECT.md
                client: reqwest::Client::new(),
            }
        }
    }
    ```

    Implement `TrackerAuth`:

    - `generate_auth_url()`:
      1. Generate `code_verifier`: random 32-byte array → base64url-encoded (no padding). Use `rand::thread_rng().gen::<[u8; 32]>()` and `base64::Engine::encode()` with `base64::engine::general_purpose::URL_SAFE_NO_PAD`.
      2. Generate `state`: random 32-byte array → hex-encoded via `hex::encode()`. This state string keys the PkceSession in TrackState.
      3. Compute `code_challenge`: SHA-256 hash of code_verifier, then base64url-encode (no padding). Use `sha2::Sha256` and `base64` engine URL_SAFE_NO_PAD.
      4. Build URL: `https://myanimelist.net/v1/oauth2/authorize?response_type=code&client_id={client_id}&redirect_uri={redirect_uri}&code_challenge={code_challenge}&code_challenge_method=S256&state={state}`
      5. Return `(url, state)` — the caller stores `state` in TrackState with the `code_verifier` for later exchange.

    - `exchange_code(code, code_verifier)`:
      1. POST to `https://myanimelist.net/v1/oauth2/token` with form body:
         ```
         client_id={client_id}
         &grant_type=authorization_code
         &code={code}
         &code_verifier={code_verifier}
         ```
      2. Use `reqwest::Client::post().form(&params)` for URL-encoded form data.
      3. Parse JSON response into `AuthToken`.
      4. Set `created_at` to current unix epoch for expiry tracking.

    - `refresh_token(token)`:
      1. POST to `https://myanimelist.net/v1/oauth2/token` with form body:
         ```
         client_id={client_id}
         &grant_type=refresh_token
         &refresh_token={token.refresh_token}
         ```
      2. Parse response into new `AuthToken`.
      3. Update `created_at` timestamp.

    - `verify_token(token)`:
      1. GET `https://api.myanimelist.net/v2/users/@me` with `Authorization: Bearer {token}`.
      2. MAL requires `X-MAL-CLIENT-ID` header as well.
      3. Parse JSON, return user info JSON string.

    Helper: Create a SHA-256 + base64url utility:
    ```rust
    fn generate_pkce_challenge(verifier: &str) -> String {
        use sha2::{Sha256, Digest};
        use base64::Engine;
        let hash = Sha256::digest(verifier.as_bytes());
        base64::engine::general_purpose::URL_SAFE_NO_PAD.encode(hash)
    }
    ```

    Helper: Generate random bytes:
    ```rust
    use rand::Rng;
    fn random_hex_string() -> String {
        let bytes: [u8; 32] = rand::thread_rng().gen();
        hex::encode(bytes)
    }
    ```

    Check if `hex` crate is available. If not, add `hex = "0.5"` to server's `Cargo.toml`. Also add `rand` if not a transitive dep that's usable (it is, via tokio, but make it explicit: `rand = "0.8"`).

    - Add `hex = "0.5"`
